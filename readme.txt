Turn your website into a Search Engine MAGNET like Quora/ Yahoo Answers.

== Description ==

If you've ever wondered why websites like Quora/ Yahoo answers and other forums 
rank so high in search engines to millions of queries, then the answer to that question 
is good quality useful content. These websites rank on top because of a simple fact 
that people get value from visiting these websites, and the search engines know that. 
If you are looking for a long term plan for higher SEO rankings, then QBOT is a tool which can 
help you achieve just that. Quality Content IS, and always will be KING.

== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "QBOT".
3. Click install.
4. Click activate.

== Screenshots ==

1. Quick dashboard to navigate through question answer features.
1. Easy to post questions from frontend by users & backend by admin.
2. Easy to answer question from backend and frontend.
3. General settings for complete control of frontend
4. Frontend question listing along with Liked, Favorite, viewed and answers count
5. Answer view page where users post answer on particular question.
